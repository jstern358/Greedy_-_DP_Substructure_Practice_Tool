DP Optimal Substructure Generator

A static, seedable multiple-choice generator for dynamic programming problems on integer lists. Problems are composed from objectives, subsequence patterns, spacing rules, and optional split/region wrappers.

Generator Settings

What each level can generate
  • Level 1: max-sum contiguous, max-sum spaced, LIS, max-sum increasing, LAP, LPS, LRS.
  • Level 2: spacing added to compatible subsequence families, plus positive max-product forms.
  • Level 3: split into k contiguous regions and maximize the worst regional optimum.
  • Level 4: regional problems where each region must use equal subsequence length.
  • Level 5: regional balancing objective: minimize max regional value minus min regional value.

This prototype intentionally reports only “correct” or “incorrect,” with no explanation after selection.
